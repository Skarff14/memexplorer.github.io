Commands

Command Description
wii bio set your bio
wii help sends you a list of the bot's commands
wii invite sends you an invite link for the bot
wii leaderboard displays the users with the most Wii Points
wii meme sends a meme
wii ping sends the bot's ping
wii popular displays the posts with the most upvotes
wii profile displays the user's profile
wii stats displays the bot's stats
wii upload upload a meme to the database
